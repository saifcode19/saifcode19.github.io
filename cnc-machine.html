cnc machine

Problem

Kids lose interest in traditional learning methods.

Solution

An interactive robotic tree that talks and moves using servo motors.

Hardware Used

  • Arduino Nano
  • Servo Motors
  • DFPlayer Mini
  • Speaker
  • Power Supply

Working

Servo motors control movement while audio module plays responses. System reacts to user interaction.

Challenges

Power instability causing servo jitter. Solved using stable power supply.

Result

Interactive learning robot for students.
