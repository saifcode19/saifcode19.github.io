EVA – Smart Educational Robot

Interactive voice assistant robot for student learning

Overview

EVA is a voice-based robotic assistant designed to improve student engagement through interactive learning.

Features

  • Voice interaction
  • Educational responses
  • Servo-based movements
  • Student engagement system

Hardware Components

  • Arduino / ESP
  • Servo Motors
  • Microphone Module
  • Speaker

Working

Voice input is processed and converted into responses. Servo motors provide physical interaction.

Demo Video-1

Challenges

  • Voice recognition accuracy
  • Audio output clarity
  • Synchronization

Result

A functional robot capable of interacting with students and assisting in learning.
